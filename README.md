## Задача 1

### Текст задачи

Сформировать список из чисел, противоположных вводимым значениям

### Алгоритм решения

Программа начинает свое выполнение с функции main. Выводится текст в консоль о вводе кол-ва чисел в списке.
Вводим целое число и оно сохраняется в значение count. Создаем список ListZ и сразу же для него вызывается
функция а с параметром z. Функция получается наше значение count, создается пустой список с помощью генератора
списка [...]. Запускается цикл for от 1 до z. В самом цикле for выводится сообщение о том, что нас просят ввести
число для формирования списка с противоположным значениями. В значение ch вводим целые числа. В список подаются
противоположные значения с помощью оператора yield (1 = -1 и тд). После завершения цикла, сформированный список
возвращается в главную функцию main. Полученный список сохраняется в ListZ, выводится сообщение "Список: "
и выводится содержимое списка listZ.

## Задача 2

### Текст задачи

Найти количество цифр в записи натурального числа

### Алгоритм решения

Программа начинается в функции main. Выводится сообщение и пользователь вводит целое число, которое сохраняется в значение x. Затем вызывается рекурсивная функция counting с параметрами z c. Функция работает так: если текущее число z равно 0, то возвращается значение счетчика c, но если исходное число было 0, возвращается 1. Если z не равно 0, функция вызывает саму себя с аргументами z/10 (отбрасываем последнюю цифру) и c+1 (увеличиваем счетчик). Рекурсия продолжается, пока число не станет равным нулю. После получения результата, результат сохраняется в значение res. Программа сообщения и количество цифр на экран.

## Задача 3

### Текст задачи

Создайте собственные функции для выполнения основных операций над списками (добавление/ удаление/поиск элемента, сцепка двух списков, получение элемента по номеру).

### Алгоритм решения

Программа начинается в функции main. Сначала пользователю предлагается ввести количество чисел для первого списка, введенное значение сохраняется в значение count. Затем вызывается функция creat с параметром count, которая создает список List1: в цикле от 1 до count пользователь вводит числа, и каждое число добавляется в список. После этого на экран выводится список List1.
Затем аналогично создается второй список: пользователь вводит количество чисел для второго списка (сохраняется в count1), вызывается функция creat с параметром count1 для создания списка List2, и результат выводится на экран.
Далее происходит сцепление двух списков с помощью функции joinz: списки List1 и List2 объединяются с помощью оператора @, результат сохраняется в список list3 и выводится на экран.
После этого выполняется добавление элемента в список: вызывается функция add с параметром List1, которая запрашивает у пользователя число, добавляет его в начало списка и возвращает новый список List4. Новый список выводится на экран.
Затем выполняется поиск элемента в списке list3: пользователь вводит значение для поиска (сохраняется в elem), вызывается функция search_el с параметром elem и list3. Функция search_el использует List.contains для проверки наличия элемента и выводит соответствующее сообщение.
Далее пользователь вводит номер элемента (индекс) для получения значения из списка list3, значение сохраняется в index. Вызывается функция number_el с параметром index и list3, которая проверяет корректность индекса и выводит элемент с указанным номером или сообщение об ошибке.
В конце выполняется удаление головы (первого элемента) из списка list3: выводится исходный список, вызывается функция del_head с аргументом list3, которая удаляет первый элемент и возвращает новый список. Результат сохраняется в ту же переменную list3 (переопределяется) и выводится на экран.
Программа завершается возвратом 0
